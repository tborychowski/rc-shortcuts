rc-shortcuts
=========

Inspired by https://git.kolab.org/diffusion/RPK/browse/master/plugins/kolab_shortcuts/.
Works with the latest RoundCube version (v1.4.9).

### Basic shortcuts
- <kbd>i</kbd> - Go to **i**nbox
- <kbd>u</kbd> - **U**pdate message list
- <kbd>n</kbd> - Compose **n**ew message
- <kbd>ctrl+c</kbd> -  **C**lose a message (in view and edit mode)
- <kbd>#</kbd> - Delete a selected message
- <kbd>e</kbd> - **E**dit a message (for draft messages)
- <kbd>s</kbd> - Focus **s**earch input
- <kbd>j</kbd> - Next message
- <kbd>k</kbd> - Previous message
- <kbd>r</kbd> - **R**eply to a selected message
- <kbd>a</kbd> - Reply **a**ll to a selected message
- <kbd>f</kbd> - **F**orward a selected message
- <kbd>cmd + enter</kbd> - Send the currenlty edited message
